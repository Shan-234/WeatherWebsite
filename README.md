# Basic Weather Website:

## ☀️ API Services:

- **AccuWeather API**  
  ➡️ Provides data about the weather at a specific location.  
  ➡️ Uses a cache to store previous requests.

## 📚 Libraries:

- **Leaflet.js**  
  ➡️ Displays map for user.

- **Chart.js**  
  ➡️ Represents data through bar/line charts.
